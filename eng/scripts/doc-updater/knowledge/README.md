# Package Knowledge Bases

This directory contains auto-generated knowledge base files produced by the knowledge-build phase of the doc-updater pipeline.

## Files

- `<package>.md` — Structured knowledge base for a package (decorators, types, feature areas)
- `<package>.meta.json` — Metadata tracking the last analyzed commit

## Usage

```bash
# Build knowledge for a specific package
npx tsx src/index.ts --config knowledge < package > --phase

# Force a full rebuild (ignore incremental)
npx tsx src/index.ts --config knowledge --full-rebuild < package > --phase
```

## Important

Do **not** edit these files manually — they will be overwritten on the next knowledge build run.

The knowledge base is generated by the `knowledge-build` phase, which uses an LLM to analyze
the package source code and produce a structured catalog. The `doc-update` phase then consumes
the knowledge base to update documentation without re-analyzing source code.
