# `@@clientOption` responseEnvelopeName renames the response envelope type

## TypeSpec

```tsp
@service(#{ title: "ClientOption" })
namespace ClientOption;

model Person {
  name: string;
  age: int32;
}

@get
@route("/list")
op list(): Array<Person>;

#suppress "@azure-tools/typespec-client-generator-core/client-option" "response envelope has custom name"
@@clientOption(list, "responseEnvelopeName", "RenamedResponse", "go");
```
