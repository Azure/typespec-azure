# Spreading a model into an operation flattens its properties into positional and optional method parameters

## TypeSpec

```tsp
@service(#{ title: "Regressions" })
namespace Regressions;

model InnerSpreadParam {
  name: string;
  code: int32;
}

model SpreadParams {
  name: string;
  color?: string;
  inner?: InnerSpreadParam;
}

@route("/spread-with-model")
op spreadWithModel(...SpreadParams): void;
```
