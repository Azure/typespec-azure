# Non-string path parameters with client-side default values are generated with their default constants

## TypeSpec

```tsp
@service(#{ title: "Regressions" })
namespace Regressions;

union Color {
  red: "red",
  green: "green",
  blue: "blue",
}

@post
@route("/non-string-path-params/{color}/{count}")
op withClientDefaultValues(
  @Azure.ClientGenerator.Core.Legacy.clientDefaultValue(Color.blue)
  @path
  color: Color,

  @Azure.ClientGenerator.Core.Legacy.clientDefaultValue(123)
  @path
  count: int32,
): void;
```
