# `@@clientOption` monomorphicResponseFieldName renames a monomorphic response field

## TypeSpec

```tsp
@service(#{ title: "ClientOption" })
namespace ClientOption;

model Person {
  name: string;
  age: int32;
}

@get
@route("/list")
op list(): Array<Person>;

#suppress "@azure-tools/typespec-client-generator-core/client-option" "response has custom field name Users"
@@clientOption(list, "monomorphicResponseFieldName", "Users", "go");
```

## The monomorphic response field is named `Users` instead of the default
