# Two clients whose names differ only by a trailing `Server` word get distinct fake interceptors

Regression test: the fake for the `Authorization` client is named `AuthorizationServer` and the
fake for the `AuthorizationServer` client is named `AuthorizationServerServer`. These two fake
types previously produced the same transport-interceptor variable name, causing a duplicate
declaration in the `fake` package.

## TypeSpec

```tsp
@service(#{ title: "Auth" })
namespace Auth;

@route("/authorizations")
interface Authorization {
  @get
  get(): void;
}

@route("/authorizationServers")
interface AuthorizationServer {
  @get
  get(): void;
}
```

## Emitter configuration

```yaml
generate-fakes: true
```
