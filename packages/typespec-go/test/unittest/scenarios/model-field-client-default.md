# A model field with a client-side default that is an extensible enum accepts values outside the predefined set

## TypeSpec

```tsp
@service(#{ title: "Regressions" })
namespace Regressions;

union Color {
  red: "red",
  green: "green",
  blue: "blue",
}

union NetworkVersion {
  string,
  v1: "2020-11-01",
  v2: "2022-11-01",
}

model Widget {
  name: string;

  @Azure.ClientGenerator.Core.Legacy.clientDefaultValue(Color.green)
  color: Color;

  @Azure.ClientGenerator.Core.Legacy.clientDefaultValue("2026-04-15-preview")
  networkVersion?: NetworkVersion;
}

@put
@route("/model-with-default-const")
op withClientDefaultModelField(@body widget: Widget): void;
```
